# hub-api

> HUB API ACCOUNT

Simply setup a new rails 5 Rest Api project
Ruby 2.3.3
Rails 5.2

# SIMPLE RAILS 5 REST API

Example of a basic REST API to perform Bank Account. Uses MySQL by default. Please add your own database and credentials. in 
config/database.yml

set your credentials...
  -username: #######
  -password: #######
  -host: ##########
  
## CLONE PROJET
git clone https://github.com/DataPartnerBr/hub-api.git

cd hub-api

## INSTALL DEPENDENCIES
```bash
bundle install
```
## CREATE DATABASE
```bash
rails db:create
```

## RUN MIGRATION
```bash
rails db:migrate
```

## RUN SERVER
```bash
rails s
```

# OPEN BROWSER
http://localhost:3000/


and ROOOORRR!!
